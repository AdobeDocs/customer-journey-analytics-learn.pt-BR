# Arquitetura e integrações da Customer Journey Analytics

Saiba como a Adobe Customer Journey Analytics foi projetada no Adobe Experience Platform para potencializar a jornada e a análise entre canais. Este vídeo explica os principais conceitos do CJA, como conexões, visualizações de dados e compilação de dados, além de como o CJA se integra ao Real-Time CDP, Journey Optimizer e ferramentas de BI para obter insights completos de jornada do cliente.

Para obter mais informações sobre o Customer Journey Analytics, consulte a [documentação](https://experienceleague.adobe.com/pt-br/docs/analytics-platform/using/cja-landing){target="_blank"}.
