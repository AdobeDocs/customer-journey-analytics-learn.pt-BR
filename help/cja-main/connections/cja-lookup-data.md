# Pesquisas do CJA para dados de perfil e pesquisa

Com as pesquisas do CJA sobre dados de perfil e pesquisa, os usuários podem utilizar relacionamentos mais profundos entre elementos de dados, como pessoa para conta em casos de uso B2B.  Este vídeo aborda como adicionar dados de pesquisa a conjuntos de dados de perfil e pesquisa, e como planejar e mapear os seus principais campos.  Lembre-se de que, se houver uma relação de um para muitos entre a chave e o valor da pesquisa, o valor mais recente será usado.

Para conferir a documentação detalhada, acesse [Tipos de conjunto de dados](https://experienceleague.adobe.com/docs/analytics-platform/using/cja-connections/create-connection.html?lang=pt-br#dataset-types){target="_blank"} e [Adicionar dados na camada das contas como um conjunto de dados de pesquisa](https://experienceleague.adobe.com/docs/analytics-platform/using/cja-usecases/b2b/b2b.html?lang=pt-br){target="_blank"}.
