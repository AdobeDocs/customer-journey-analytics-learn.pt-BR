# Relatório em tempo real

Descubra como os relatórios em tempo real no Adobe Customer Journey Analytics fornecem métricas principais como receita, conversões e visitas diretamente no Analysis Workspace. Ideal para grandes promoções, lançamentos de produtos e validação de jornadas, esse recurso permite que os profissionais de marketing tomem decisões mais rápidas e orientadas por dados quando cada minuto é importante.

Para mais informações, consulte a [documentação](https://experienceleague.adobe.com/en/docs/analytics-platform/using/cja-components/real-time-reporting/real-time){target="_blank"}.
