# Servidor MCP remoto para [!DNL Customer Journey Analytics] em ação

Saiba como conectar ferramentas de IA, como Claude, ChatGPT ou Cursor, ao servidor MCP do Adobe Customer Journey Analytics (CJA) para explorar visualizações de dados, executar relatórios e gerar insights de KPI prontos para executivos usando linguagem natural. Esta demonstração mostra como os servidores MCP remotos permitem fluxos de trabalho de análise escaláveis e controlados e relatórios automatizados no Adobe Experience Platform.

Para mais informações, consulte a [documentação](https://developer.adobe.com/analytics-mcp/docs/cja/){target="_blank"} do Content Analytics.
