# [!DNL Content Analytics] - Editar configuração

Edite configurações do [!DNL Content Analytics] com eficiência ajustando nomes, visualizações de dados e configurações de captura de experiência. Essa ação atualiza métricas automaticamente e executa o ajuste por meio da extensão Tags do Content Analytics para obter insights otimizados.

Para obter mais informações, consulte a [documentação](https://experienceleague.adobe.com/en/docs/analytics-platform/using/content-analytics/configuration/guided){target="_blank"}.
