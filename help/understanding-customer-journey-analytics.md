# Noções básicas do Customer Journey Analytics

O Adobe Customer Journey Analytics (CJA) é um serviço de aplicativos criado na Adobe Experience Platform. Ele traz a ferramenta de análise avançada conhecida como Analysis Workspace para o Platform, para permitir que você faça a análise de vários canais em qualquer um dos conjuntos de dados do Platform.

>[!VIDEO](https://video.tv.adobe.com/v/30090/?quality=12&enable10seconds=on&speedcontrol=on)

Para obter mais informações, consulte a [documentação](https://docs.adobe.com/content/help/pt-BR/analytics-platform/using/cja-landing.html).
